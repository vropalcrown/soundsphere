# **App Name**: AudioSplit

## Core Features:

- Device Detection: Automatically detect and list available audio output devices connected to the PC.
- Multi-Device Selection: Allow users to select multiple audio devices to create an output group for simultaneous audio routing.
- Individual Volume Control: Provide individual volume controls for each selected device within the output group.
- Configuration Saving: Remember user-defined audio output configurations and reload them.
- Intelligent Volume Suggestion: Intelligently tool to suggest initial volume settings based on the device types connected, like speaker and headphones.

## Style Guidelines:

- Background color: Dark gray (#222222) to reduce eye strain and provide a professional look.
- Primary color: A vibrant blue (#3498db) to provide a visual anchor within the dark theme.
- Accent color: Light blue (#85c1e9), an analogous hue to the primary, for interactive elements and highlights to ensure visibility against the dark backdrop.
- Body and headline font: 'Inter', a grotesque sans-serif suitable for both headlines and body text, ensuring legibility and a modern feel.
- Use clear and modern icons for audio devices and controls to enhance usability.
- A clean and intuitive layout for easy management of audio devices and volume levels.
- Subtle animations for volume adjustments and device selection to provide user feedback.
